Animated-Stats

Animated-Stats is a dynamic, Pokémon-inspired GitHub profile card generator. It fetches live GitHub stats for any user and displays them in a stylish, animated card with customizable themes and type icons.

What Does It Do?

  • Fetches GitHub user stats (followers, stars, public repos, top languages, recent activity)
  • Displays them in a Pokémon-style card with animated, customizable themes
  • Lets you choose the user and card style via URL parameters

How to Use

  1. Open the site in your browser (e.g., https://your-vercel-url/).
  2. Add ?user=USERNAME to the URL to show any GitHub user's stats.
    Example: ?user=Leorev01
  3. Add &type=TYPE to change the card's style (see below for options).
    Example: ?user=Leorev01&type=water
  4. Share the link with others, or take a screenshot to use in your README.
Note: GitHub READMEs do not support live HTML/JS. To embed the card as an image, use a screenshot.

Card Styles

You can change the card's appearance by setting the type parameter in the URL. Supported types:

normalfirewaterelectricgrassicefightingpoisongroundflyingpsychicbugrockghostdragondarksteelfairy

Each type changes the card's color and icon to match classic Pokémon types.

Example

Example: ?user=Leorev01&type=water

How to Add to Your Profile README

  1. Take a screenshot of your card (or use a screenshot service).
  2. Upload the image to your repository or an image host.
  3. Add it to your README.md like this:
    ![My Dev Card](https://your-image-url.png)
  4. Optionally, link the image to your live card:
    [![My Dev Card](https://your-image-url.png)](https://your-vercel-url/?user=USERNAME&type=TYPE)
